# 十五、实体

|   字符   |  含义   |
| :------: | :-----: |
| `&nbsp;` | `空格`  |
|  `&#9;`  | `Tab格` |
| `&amp;`  |   `&`   |
|  `&lt;`  |   `<`   |
|  `&gt;`  |   `>`   |
| `&quot;` |   `"`   |
| `&qpos;` |   `'`   |

`&#9;`代表的 Tab 格只有在`<pre></pre>`中才生效。
