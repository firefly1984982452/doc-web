# 拖动事件

拖动目标

- `ondragstart` 开始拖动元素时
- `ondrag` 元素正在拖动
- `ondragend` 完成元素的拖动

释放目标

- `ondragenter` 拖动的元素进入放置目标
- `ondragover` 拖动元素在放置目标上
- `ondragleave` 拖动元素离开放置目标
- `ondrop` 拖动元素放置在目标区域

◆ 优先级顺序

1. `ondragstart`
2. `ondrag`
3. `ondragenter`
4. `ondragover`
5. `ondragleave`
6. `ondrop`
7. `ondragend`
