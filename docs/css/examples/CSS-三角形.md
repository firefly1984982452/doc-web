# CSS - 三角形


## 等腰三角形

```css
div {
  width: 0;
  height: 0;
  border-color: #f60 transparent transparent transparent;
  border-style: solid;
  border-width: 10px;
}
```

<div class="example-box">
  <div style="width: 0;
  height: 0;
  border-color: #f60 transparent transparent transparent;
  border-style: solid;
  border-width: 20px;"></div>
</div>

```css
div {
  display: inline-block;
  width: 0;
  height: 0;
  border-color: #fff transparent;
  border-width: 20px 0 0 20px;
  border-style: solid;
}
```
