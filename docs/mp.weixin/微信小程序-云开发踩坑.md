# 数据库

如果要导入数据时，不是一般的json格式，是以下格式：

```
{}
{}
{}
{}
```
**没有逗号，外面也不用数组包起来**


## 查
```js
const db = wx.cloud.database()
db.collection('app').limit(5).where({
    tag: 'JavaScript'
}).get({
    success: function(res){
    that.setData({
        list : res.data
    })
    }
})
```

## 增
```js
const db = wx.cloud.database()
db.collection('app').add({
    data: {
    'name': '1'
    },
    success(res) {
    console.log(res)
    },
    fail(err) {
    console.log(err)
    }
})
```
## 改
```js
const db = wx.cloud.database()
db.collection('app').doc(_id).update({
    data: {
        agreen: 1
    },
    success(res) {
        console.log(res)
    },
    fail(err) {
        console.log(err)
    }
})
```

## 删
```js
const db = wx.cloud.database()
db.collection('app').doc(_id).remove()
```
